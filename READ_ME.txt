Put All Files in same folder then run the ChessMain.py file in any text editor.
It will show the welcome interface where you can select whether you want to play two player or with computer.
If you select player vs player option, then you can play turn by turn.
If you select player vs computer, then you have to select difficulty level and then select whether you want to play with black or white.
All the chess rules like castling, en-passant and Pawn Promotion(With options) have been considered while defining the moves.
Sounds are also added whenever there is a move,capture,check,checkmate or stalemate.
The engine is also properly calculating good moves, captures and checkmate protection.
It also considers whether there are enough pieces to checkmate, otherwise it shows Insufficient Material.
